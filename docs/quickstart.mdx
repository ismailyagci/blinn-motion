---
title: "Quickstart"
description: "Install a Blinn Motion adapter and play your first MotionDoc in three steps."
---

You need two things to render an animation: a **MotionDoc** (the JSON document) and a
**host** (element, component, or custom element). Here's the shortest path from zero to motion.

<Steps>
  <Step title="Install an adapter">
    Pick the package for your stack. Each web binding pulls in `@blinn-motion/core` and the paint backends it needs.

    <CodeGroup>
    ```bash DOM
    npm i @blinn-motion/dom
    ```
    ```bash React
    npm i @blinn-motion/react
    ```
    ```bash Vue
    npm i @blinn-motion/vue
    ```
    ```bash React Native
    npm i @blinn-motion/react-native
    ```
    </CodeGroup>

    Also available: `@blinn-motion/canvas`, `@blinn-motion/svelte`, `@blinn-motion/angular`, `@blinn-motion/lit`.
  </Step>

  <Step title="Get a MotionDoc">
    Either **export one from Figma** with the [Blinn Motion plugin](/guides/figma-plugin)
    (Export → pick your framework → download JSON), or
    [hand-author one](/concepts/motiondoc) for tests. A MotionDoc is plain JSON:

    ```jsonc card.motion.json
    {
      "format": "motion-engine",
      "version": "1.0",
      "duration": 1.2,
      "fps": 60,
      "stage": { "width": 375, "height": 600, "background": "#0E1116FF" },
      "layers": [ /* … */ ]
    }
    ```
  </Step>

  <Step title="Mount it and press play">
    Hand the engine your host and the document.

    <CodeGroup>
    ```ts DOM
    import { create } from "@blinn-motion/dom";
    import doc from "./card.motion.json";

    const player = create(
      document.getElementById("stage")!,
      doc,
      { loop: true, autoplay: true }
    );

    player.play();
    player.seek(0.8);
    player.setProgress(0.5); // 0…1 — scroll / gesture
    ```
    ```tsx React
    import { BlinnMotion } from "@blinn-motion/react";
    import doc from "./card.motion.json";

    export default function Hero() {
      return <BlinnMotion doc={doc} renderer="dom" loop autoplay />;
    }
    ```
    ```vue Vue
    <script setup>
    import { BlinnMotion } from "@blinn-motion/vue";
    import doc from "./card.motion.json";
    </script>

    <template>
      <BlinnMotion :doc="doc" renderer="dom" :loop="true" :autoplay="true" />
    </template>
    ```
    </CodeGroup>
  </Step>
</Steps>
