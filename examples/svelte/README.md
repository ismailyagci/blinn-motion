# Blinn Motion · Svelte example

**Stack:** Svelte 5 + Vite + `@blinn-motion/svelte` (`use:blinnMotion` action)

Advanced shared demo flow: MotionDoc switch, dual DOM/Canvas, transport, scrub, rate, progress mode, live meter.
