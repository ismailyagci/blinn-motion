# Blinn Motion · Expo example

**Stack:** Expo + `@blinn-motion/react-native`

Advanced native demo: MotionDoc switch, transport, scrub, rate, progress-driven mode, onFrame meter.
No dual Canvas stage (RN paints Views only).
