# Blinn Motion · Vanilla example

**Stack:** Vite + TypeScript · `@blinn-motion/dom` + `@blinn-motion/canvas`

Advanced shared demo flow without a framework: MotionDoc switch, dual DOM/Canvas
stages, transport, scrub, rate, progress-driven mode, live time/fraction.

| | |
|--|--|
| 📚 Docs | [DOM adapter](https://docs.blinnmotion.com/adapters/dom) · [Canvas adapter](https://docs.blinnmotion.com/adapters/canvas) |
| 🔴 Live | [vanilla.blinnmotion.com](https://vanilla.blinnmotion.com) |

```bash
npm install
npm run dev
npm run build
```
